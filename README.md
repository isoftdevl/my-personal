# My Personal Portfolio Website

This repository contains the source code for my professional personal portfolio website. It is designed to showcase my expertise in backend systems architecture, Laravel development, and modern web technologies.

## 🌐 Live Demo
Checking out the live site: [isoftdevl.github.io/my-personal](https://isoftdevl.github.io/my-personal) *(or your deployed URL)*

## 🚀 Built With
- **Frontend**: React.js v18 with TypeScript
- **Bundler**: Vite
- **Animations**: Framer Motion
- **Styling**: Vanilla CSS (Modern Dark/Neon Theme)
- **Icons**: Lucide React

## ✨ Key Features
- **Project Spotlight**: Detailed breakdowns of my senior-level backend contributions (OrickFit, B-Solar Pro, etc.).
- **CV Modal**: Interactive, professional CV overlay with print/download functionality.
- **Responsive Architecture**: Fully optimized for mobile, tablet, and desktop viewing.
- **Performance**: High-speed loading and smooth transitions.

## 🛠 Local Development
To run this project locally:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/isoftdevl/my-personal.git
   ```
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Start the development server**:
   ```bash
   npm run dev
   ```
